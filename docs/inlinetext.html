Inline Markup¶

Inline markup allows words and phrases within text to have character styles (e.g. italics and boldface) and functionality (e.g. hyperlinks).

Be aware of some restrictions of this markup:

  • It may not be nested.

  • Content may not start or end with whitespace: * text* is wrong.

There are nine inline markup constructs. Four common constructs are given below.

Descripion

emphasis

Normally rendered as italics.

Escaping characters¶

If you want to display a character with its normal meaning, which would otherwise create inline markup, you must escape the character by prepending it with a backslash (\).

For example, to show how to specify the appropriate markup characters for the emphasis and strong constructs dsplayed above, you include a backslash before each markup character.

  • Emphasis construct - Type \*emphasis\*, which renders as *emphasis* instead of emphasis.

  • Strong construct - Type \*\*strong\*\*, which renders as **strong** instead of strong.
